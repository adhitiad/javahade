Kebijakan Privasi

Privasi Anda adalah prioritas mutlak kami. Di Javahade, kami berkomitmen untuk melindungi data pribadi dan sesi privat Anda.

  • Data Enkripsi: Semua lalu lintas web dan pesan instan Anda dienkripsi secara penuh (*end-to-end*).
  • Sesi Privat: Tidak ada sesi *Live Booking* yang direkam oleh server kami tanpa izin eksplisit dari Host dan Klien.
  • Data Biometrik (KYC): Foto KTP dan *selfie* Anda diproses oleh sistem AI (Groq Vision) untuk keperluan keamanan transaksi dan menghindari bot. Kami tidak memperjualbelikan data wajah Anda kepada pihak ketiga mana pun.
